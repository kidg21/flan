# Media Block

#### A Media Block positions media alongside descriptive content.

<Preview withToolbar>
  <Story name="Standard" decorators={[CenterDisplayGrid]}>
    <Card padding="4x" shadow="2x">
      <MediaBlock
        media={ModernExterior2}
        title="Media Block"
        description="In life you need colors. We'll put a happy little sky in here."
      />
    </Card>
  </Story>
</Preview>

# Usage

The Media Block supports a Body (Title and Description) and a Media element.  A typical media element would be an image or an icon but any media source can be passed to the `media` prop.
