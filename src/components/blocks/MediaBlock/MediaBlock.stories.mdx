# Media Block

#### An Media Block allows users to toggle the display of sections of content

  <Story name="Standard" decorators={[CenterDisplayGrid]}>

      />
    </Card>
  </Story>

# Usage

The MediaBlock component is a set of expandable panels for showing and hiding content from the user to make the UI less crowded. The component works well on both mobile and desktop environment and it supports keyboard and screen readers too. The component supports Lumo and Material themes.
