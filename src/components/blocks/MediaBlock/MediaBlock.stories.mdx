# Media Block

#### A Media Block positions media alongside descriptive content.

<Preview withToolbar>
  <Story name="Standard" decorators={[DisplayCard, CenterDisplayGrid]}>
    <MediaBlock
      title="Media Block"
      description="The body content goes here.  The media content is over there. â†’"
      media={ModernExterior1}
    />
  </Story>
</Preview>

# Usage

The Media Block supports a Body (Title and Description) and a Media element.  A typical media element would be an image or an icon but any media source can be passed to the `media` prop.

# Examples

### Knobs
To configure this component, go to the `Canvas` tab and navigate to the `Knobs` option on the `Addons` panel.

<Preview withToolbar>
  <Story name="Knobs" decorators={[DisplayCard, CenterDisplayGrid]}>
    <MediaBlock
      title={text("Title", "Media Block", "Media Block")}
      description={text("Description", "The body content goes here.  The media content is over there. â†’", "Media Block")}
      media={select("Media", [null, ModernExterior1, Spiderman], ModernExterior1, "Media Block")}
      icon={select("Icon", [null, "user_circle", "bookmark"], null, "Media Block")}
      mediaSquare={boolean("Square", false, "Media Block")}
      mediaRound={boolean("Round", false, "Media Block")}
      mediaReverse={boolean("Reverse", false, "Media Block")}
      size={select("Size", ["xs", "sm", "default", "lg", "xl", "2xl"], "default", "Media Block")}
    />
  </Story>
</Preview>
