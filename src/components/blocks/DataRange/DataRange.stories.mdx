# DataRange

### An DataRange allows users to toggle the display of sections of content

# Usage

The DataRange component is a set of expandable panels for showing and hiding content from the user to make the UI less crowded. The component works well on both mobile and desktop environment and it supports keyboard and screen readers too. The component supports Lumo and Material themes.
