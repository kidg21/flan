# Data Range

#### A Data Range component shows the range between defined values.

<Preview withToolbar>
  <Story name="Standard">
    <DataRange
            id="default"
            label="Data Range (default)"
            helpText="Hang in there, buddy, I'm here to help!"
            min={{
              label: "Min",
            }}
            max={{
              label: "Max",
            }}
          />
  </Story>
</Preview>

# Usage

The Data Range component allows a user to enter or select numeric values to filter data by. It is an alternative to using the Slider component.
