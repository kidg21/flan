# Usage

Segmented Controls can be used in replacement of a Switch, radio buttons, or checkboxes depending on how you set them up.

- **Single select** : One option is always selected.
- **Toggle** : All options are optional. All may be selected or all may be unselected.

Segment controls should not be used for large amounts of options. Two or three options is best practice.

# Examples

### States

You can manage the stage of controls by each individual control item or the entire control group.
An individual control item can be selected, unselected, or disabled.
A control group has the capability to override all control items if it is set to disabled, as seen in the second example below.
