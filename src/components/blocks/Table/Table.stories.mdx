# Table

### A table displays sets of data across rows and columns.

<!-- <Story name="Simple" parameters={{ decorators: [Container] }}>{stories.Simple()}</Story> -->

# Usage

Tables display information in a grid-like format of rows and columns. They organize information in a way that’s easy to scan, so that users can look for patterns and insights.

Tables can contain:

- Interactive components (such as Commands, Buttons, or Select Menus)
- Non-interactive elements (such as Badges)
- Tools to query and manipulate data
