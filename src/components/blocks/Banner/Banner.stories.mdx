# Banner

### An Banner allows users to toggle the display of sections of content

<Preview withToolbar>
  <Story name="Standard">
     <Banner title="This is a Banner." />
  </Story>

# Usage

The Banner component is a set of expandable panels for showing and hiding content from the user to make the UI less crowded. The component works well on both mobile and desktop environment and it supports keyboard and screen readers too. The component supports Lumo and Material themes.
