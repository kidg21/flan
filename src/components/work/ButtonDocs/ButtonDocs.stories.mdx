## Stories

With `MDX` we can define a story for `Button` right in the middle of our markdown documentation.

### Standard

This is a standard `Button` component

<Story name="Standard">
  <Button label="Standard Button" />
</Story>

### Type

Button `Type` can be displayed in a `Preview` to get a code snippet:

<Preview>
  <Story name="Types">
    <Grid>
      <Button label="Standard Button" />
      <Button label="Solid Button" type="solid" />
      <Button label="Inline Button" type="inline" />
      <Button label="Underlined Button" type="underlined" />
    </Grid>
  </Story>
</Preview>
