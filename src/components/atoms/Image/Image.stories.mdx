# Usage

Unless a size is specified, images will fill the size of its container.

Images can represent the content or be used as an element of other components. Some examples include, a **[Card](/?path=/docs/elements-card--media)** or a **[Media Block](/?path=/docs/blocks-media-block--documentation)**.
