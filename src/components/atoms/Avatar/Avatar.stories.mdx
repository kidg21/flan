# Avatar

### An Avatar represents a person or an object.

<Preview withToolbar>
  <Story name="Standard">
    <Avatar label="LG" />
  </Story>
</Preview>

# Usage

Avatars support either a label or an **[Icon](/?path=/docs/atoms-icon--documentation)**. They can be displayed alone or with other components, including:

- **[List](/?path=/docs/blocks-list--documentation)**

# Examples

### Labels

A 2-character label can be defined using the 'label' prop.

  <Story name="Labels">

### Colors

Default Avatar color is a light version of our primary color. Primary and secondary colors are supported in addition to our funciontal colors.

<Preview>
  <Story name="Colors">
    <Grid columns="4">
      <Avatar label="DF" />

      <Avatar label="SC" color="secondary" />
      <Avatar label="IN" color="info" />
      <Avatar label="SC" color="success" />
      <Avatar label="WN" color="warning" />
      <Avatar label="AL" color="alert" />
    </Grid>
  </Story>
</Preview>

### Icons

An Avatar can be configured to use any icon from our **[Icon Library](/?path=/docs/application-libraries-icon-library--app-icons)**.
