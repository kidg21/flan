# Calendar

### The Calendar is an input field that allows you to select dates, times, or date/time through textual input or mouse interaction with a calendar drop down.

# Usage

The calendar component displays a simple calendar that can be:

- Used to display a calendar of dates.
- Composed with other components to build a datepicker.
- Used to display a time (hour, minute, AM/PM).
- Used to display both date and time.

# Examples

### Date Picker

The Date Picker provides the user with a text input in which they can input month/day/year. Simple date pickers are typically used when the date is known by the user, such as a birthday or credit card expiration.
