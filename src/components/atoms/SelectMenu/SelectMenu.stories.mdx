# Select Menu

### Select is a type of input that is used in forms, where a user is submitting data and chooses one option from a list.

<Preview withToolbar>
  <Story name="Standard">
    <SelectMenu
      label="Select Label"

      ]}
    />
  </Story>
</Preview>

# Usage

Use Select Menu component to select value from options.

Select Menu should be used in conjunction with the Form from Layout. Select Menu allows for a condensed way to show options.
