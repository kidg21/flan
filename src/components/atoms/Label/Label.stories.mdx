# Usage
Labels are used to identify an input or component. They will always be situated above or next to another component in order to help ientify that component's subject or meaning. They are typically used in conjunction with the following components:

- Radio
- Checkbox
- Switch
- Text Input
- Range Slider

# Guidelines
- **Dependent:**
  Labels should be paired with another component that needs additional identification or clarification.
- **Findable:**
  Labels should be easy to find among other elements, including other Labels.
- **Clear:**
  A Labelâ€™s action and state should be clear.
