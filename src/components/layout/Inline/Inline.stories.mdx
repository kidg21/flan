# Inline

#### Inline is used to align a sent of elements inline next to each other

<Preview withToolbar>
  <Story name="Standard">
    <Inline>
      <Tag label="Tag 1" />
      <Tag label="Tag 222222" />
      <Tag label="Tag 33333333333" />
      <Tag label="Tag 444444444444444444" />
    </Inline>
  </Story>
</Preview>

# Usage

If you have a set of elements that you wish to inline, make them children of this component.

You can:

* Control horizontal alignment with `contentJustify` of `left` (default), `center` or `right`
* Control vertical alignment with `contentAlign` of `center` (default), `top` or `bottom`
* Set horizontal spacing between elements with `spacingX`
* Set vertical spacing between elements with `spacingY`

This element is a block, and it's children behave like `display: inline-block` but with flexbox-enabled alignment controls.
