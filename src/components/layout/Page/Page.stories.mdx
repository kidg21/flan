# Page

#### Used to layout 1 or more content elements within a space.
<br/>

### "Default" Pages

#### The default Page stacks content vertically within a single-column grid.

### "Configured" Pages

#### A Page can be configured to meet multiple layout needs through its 'template' prop.  Choose a template from the Template Library that will support the number of components being displayed.
<br/>

##### A Templates = 1 component, B Templates = 2 components, C = 3, D = 4, E = 5

##### Template format: [A-E]_[index] (ex. template="C_02")
