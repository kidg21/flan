Badges contain numeric values and indicate a running tally, such as the number of unread messages in a room. Use badges to label page items with a small amount of information that draws attention to new, updated or removed content. Badges are normally placed before or after the label of the thing they're quantifying. They can contain numbers, a max count (using a "+" sign to indicate a quantity greater than the displayed number), or a label to display status information.

- Use badges in conjunction with a single item so that there is no ambiguity around which item is indicated.
- Use tooltips on badges when they're useful to clarify meaning or enhance user understanding â€“ for example to indicate units.
- Badges are for counts, so they should only contain integer numbers.
- Don't use badges to visually label UI objects; use tags instead.
